# EDF Demo 脚本快速参考
# Quick Reference Card

## 基本使用 (Basic Usage)

```bash
# 运行脚本
python edf_demo.py

# 指定文件
python edf_demo.py ./edf/李诗敏1.edf
```

## 交互式菜单 (Interactive Menu)

```
1. 显示元数据 - 查看文件信息、患者信息
2. 显示通道列表 - 查看所有通道
3. 绘制波形图 - 可视化 EEG 信号
4. 退出 - 关闭程序
```

## 常见问题 (FAQ)

### Q: 如何处理中文文件名？
A: 脚本已自动处理，使用 `encoding='latin1'` 参数

### Q: 内存不足怎么办？
A: 脚本使用 `preload=False` + `crop()` 优化内存，只加载需要的部分

### Q: 如何保存图像？
A: 在绘图窗口中按 `S` 键保存图像

### Q: 支持哪些文件格式？
A: 支持 EDF、EDF+、BDF 格式

## 推荐绘图参数 (Recommended Plotting Parameters)

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 通道数 | 10 | 同时显示 10 个通道 |
| 时长 | 10 秒 | 显示 10 秒数据 |
| 缩放 | 100 µV | EEG 标准缩放 |
| 起始时间 | 0 秒 | 从文件开头开始 |

## 快捷键 (Keyboard Shortcuts)

在绘图窗口中:

## 性能提示 (Performance Tips)

1. **启动**: 使用 `preload=False` 快速查看元数据
2. **绘图**: 只加载需要的通道和时间窗口
3. **大文件**: 使用较小的时长参数 (5-10 秒)
